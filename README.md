# SkillHub

SkillHub is a full-stack online learning platform inspired by Udemy, designed with a modern and scalable architecture. The project focuses on building a production-style application using Next.js, Node.js, Prisma, and PostgreSQL.

## Tech Stack

- Next.js 16
- Node.js & Express
- TypeScript
- Prisma ORM
- PostgreSQL
- Redis
- Socket.IO
- Docker
- pnpm Workspace (Monorepo)

## Project Structure

```text
skillhub/
├── apps/
│   ├── frontend/

    ├── database/

```
